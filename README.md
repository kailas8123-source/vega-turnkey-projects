# Kacheri Sarees

A premium static storefront for handcrafted sarees and jewellery. The site includes a product-led homepage, shop filters, product details, wishlist state, cart drawer, brand story, contact form, FAQ, and newsletter signup.

## Run Locally

```bash
npm install
npm run dev
```

The local site runs at `http://127.0.0.1:5173`.

## Deploy

This project is a static HTML/CSS/JavaScript site. Vercel can deploy it directly from the repository root.
